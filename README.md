# Predictive Analytics Using Historical Data

## Project Overview
This project predicts future sales trends using historical Superstore sales data and Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Features
- Data preprocessing
- Sales trend visualization
- Linear Regression prediction
- Future sales forecasting
- Accuracy evaluation

## Output
- Predicted sales values
- Trend analysis graphs
- Correlation heatmap

Sample Superstore Dataset
